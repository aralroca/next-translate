# App dir example

Example loading and consuming translations with **Next.js 16** app dir.

## Features demonstrated:

- **Next.js 16 Support**: Uses the latest Next.js version.
- **Server Components**: Using `createTranslation` in async Server Components.
- **Client Components**: Using `useTranslation` hook in `'use client'` components.
- **Remote Loading**: Demonstrates how to use `loadLocaleFrom` to fetch translations from a remote source (mocked in this example).
- **Trans Component**: Using the `Trans` component for complex translations with HTML.
- **Turbopack**: Configured to work with Turbopack (default in Next 16).
